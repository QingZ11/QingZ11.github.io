好像有三年了，21 年知道、认识的 yihong。初识的时候，我只知道他是我们（nebula）的其中一名 contributor，至于他贡献了什么，十分抱歉，我已经忘记了。真要让 yihong 自己回想，估计也是个重大的工程吧，毕竟他的 pr 如此之多。至于为什么要采访他，进一步了解他，说实话，非常简单——他是少有的愿意接受采访的人。

其实在采访之前的沟通，我已经感受到这个人的巨大热情，他对你（旁人）的事如此的上心，不带贬义地说一句他是个好人，真的对得起“好人”这个词的原意。在后面，我收到过他发的麦当劳猫袋——番茄和葱白热衷过一段时间的毛毡包，还有今天字节的招聘信息。即便不是这些他日常留意的事物，倘若我有什么问题咨询他，他也是十分热心回复。在回复这块，我想没有人会不喜欢一个及时回复你的人，感受到了对方满满的交流诚意。——当然，信息的及时性不全代表什么，至少证明对方非常重视另外一方的心情：等待总是一种煎熬，而不是一种感情发酵。

如此热心肠的一个人，他会在开源领域得到美誉，自然是顺理成章的事情：热情、慷慨，不只是不吝啬自己的知识——开源了各种实用项目，还记得在 HG 的热点速递中推荐过不少他的开源项目，像是 xiaogpt，但我永远喜欢他 GitHub 个人主页的那个跳动的爱心。也不吝啬自己的流量——安利他中意的开源项目，作为一个粉丝不少的推主，他的安利频率如此之高，以至于他和其他的大 v 风格如此不同，很难觉得他是个大 v，也就自然没有了中不可高攀的距离感。对，他是个极具亲近感的人，如此的谦逊平和，我从未见过一个人如此地善于反思自己，似乎在网络世界（推特）上，他没有自己的脾气，即便有人蹭流量、说他是小丑，他也没有愤懑地反驳对方，反而承认对方指责的理由是对的——摊手，只能说 yihong 是个行走的大度，我没见过一个人遇到指责，下意识不是反驳，而是反思、承认对方正确的地方。

就是这样一个人，他如此的特别，我想留点我对他的印象。但他又如此的普通，估计也只有我会留点关于他的记录。

最后，我永远感谢 yihong，无论是安利过的跑步鞋，还是北京小朋友旅游攻略。像他 GitHub 跳动的那颗心一样，他的满满活力感染我，让我努力也变成一个让世界更好的人。
