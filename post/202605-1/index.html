淋雨

我不记得什么时候开始的，我很喜欢去淋雨。脑子里有的记忆，是高中的时候，在学校的电话亭打完电话，外面下起了暴雨，我慢悠悠地走回寝室；再后来就是在 SF 的时候，在暴雨中骑着车，雨水在脸上流淌，不知道是雨水还是泪水。当然我没啥好流泪的，毕竟大多时候淋雨的我，都挺快乐的。

现在，再也没有什么想要淋雨的念头。只有，淋雨了会感冒，会生病的劝解。大概，这就是岁数大了。

加班

本该和小月月吃完晚饭回来加班，但突然不想了。有一股没有意思的想法，盘踞在脑子中。加班的事情就留给明天吧。

backup

backup 好像是备份的意思，后备，应该用什么单词呢？母鸡，现在的我，在问我自己一个问题，我想要依赖大哥的程度有多高。完全没有肯定是不成立的。至少，我说出，我怕我的问题不好，到时候找你来场外求助的话，一定是动了，这是一个可以依靠下的想法。

所以，我看见说“但是我的时间有限”，才会有一种反思感。我不应该给人造成麻烦，或者说是某种程度上的心理负担。哪怕那个负担很微弱。

就像，昨天我找了 yetone 问他能不能接受专访一样。到今天中午我才知道，其实他个人拒绝了很多专访，他本身不喜欢这件事。至此，我觉得我其实给对方造成了一定程度，可能只是很轻微，或者是微乎其微的困扰。即便如此，我也觉得自己的行为，是不 ang，大概就是不对的。就像场外求助这件事一样，会造成他人困扰的请求，都是不合适的。

o.o 好好反思下。

框和边界

这是前天在上海，haoyang 的转正餐，haoyang 问了一句，你似乎有很多边框。我想回答，但我的应变能力不足以让我回复这个问题。

我现在想明白，我的所有边框，都是事情本不应如此，真实越来越难以看见。就想大众好评，打卡送吃的，好评送增评，我觉得它本不应该如此，每家商店都如此，到底哪家是真的好吃，哪家是真的不好吃，很难区分。就像我坚持，会给差评，当然我好评写的也不少。无论差评，还是好评，都是出于我真实的感受来的，并不是商家说的写个好评给 10 元返现。假如商家那么说了，我反倒会收起原本想写的好评。——这么一来，我只是在叛逆吧。

这样想来，其实我所有的原则和界限，都只有一点，它背离了真实，所以我选择反对。

我是个好人

这是一个陈述句，我是个好人。并不是思为那种好人，我很喜欢思为的人格，我没见过比他更崇高的人。但，他的好，让我觉得自己过于“我很抱歉我生而为人”，大概就是菩萨和凡人的 gap 吧。

但，即便如此，我是个好人，还是成立的。我的好人品，体现在无论这个人是我讨厌的，还是喜欢的，他/她有什么问题，我能帮上忙的，一定要帮个忙。就像隔壁组的狗屎组长，虽然我喷他 pua 实习生，但我也在他被 +1 喷的时候，给他递点选题素材。就像，我不满隔壁组的妹子，我们接了他们组的活，没有一点想要帮忙的想法，但，我见她电脑卡成狗，我会找两组共同的 +1 去要求换电脑一样。我是个好人，无关乎你是否是个好人，如果你需要帮忙，我知道，我会尽力。

这就是我是个好人的论证。其他，不重要。我有太多不喜欢和讨厌的人，但不妨碍我是个好人。

说点开心的吧

昨天吃了 DQ 抹茶冰淇淋，今天吃了优诺酸奶冰淇淋，还有野人先生抹茶冰淇淋，还有晚餐，餐馆叫啥名字，我忘记了，反正有个菜品是面包诱惑，我这两天吃了四次冰淇淋。

还有，就是我今天竟然 o.o 问国际友人能不能接受我的专访。我有些被自己惊讶到了，我竟然开了口。上面说着抱歉，但其实对 yetone 的邀约，我也觉得自己干的不错。只是就这个行为而言，但给人造成困扰也是真的。

今天一觉睡到了十一点，感觉也挺快乐的；等月月和包子来吃饭的时候，在龙湖天街负一楼买了一堆吃的，在晚饭餐馆等位区吃着关东煮和冰淇淋，也很开心；午饭煮的小馄饨也很好吃，因为小青菜煮的不错；去找小电动的路上，下起了暴雨，踩到水坑的我，也挺开心的；在骑小电动回家的路上，看见另一条车道上的车子驶过，水溅起来的水幕有两米高，我也觉得很快乐。

单独罗列一条，周二发生的，我每天晚上回家会打不到几把德州，这个数字不到两位数。周二，第一把。o.o (⁎⁍̴̛ᴗ⁍̴̛⁎) 皇家同花顺，同桌的人都惊呆了，纷纷表示都是第一次见。我也是第一次见，同花顺我拿到过，皇家同花顺第一次。那把我赢了 20k 的筹码，然后过了两把我就跑路了，离开桌子了。anyway，皇家同花顺诶 (≧∇≦) 真是开心。

再一次，我知道了，人类的赞歌，是勇气的赞歌，是 JOJO 里的台词。o.o 当然还有一句知名，可能大概是知名吧，我没和人对过这件事。那就是：JOJO，我不做人了。😂 很搞笑的台词，但被正儿八经地说出来了。

狗屎的 D

其实已经 push 到 GitHub 了，但突然想起来这件事，还是要喷一喷狗屎的 D。D 和小月月有些矛盾，双方彼此都有矛盾，也不算事小月月单方面问题。所以，小月月不遗余力地给我分享 D 的狗屎事件，然后我不遗余力地转发给大哥（因为狗屎事件和大哥有关系。

我该如何脱敏这个事情呢。真是烦人，大概就是 D 的同事说想让小客服的号加下大哥，对一下大哥在公司做的分享的文字稿。D 回了一句，大哥是看在看在她面子上才去分享的之类的话。狗屎的点在于，公私不分，我讨厌这种行为。包括，今天吃饭小月月说，D 在离职之前，用小客服号给自己的微信转了好多前司用户的微信一样。我觉得很狗屎，站在公司层面，没有讨伐的立足点，但我还是觉得狗屎。

即使我上面说了很多狗屎，我还是很佩服她的理财能力。所以，我夸她厉害，也是真心实意的，我的确觉得她理财能力很强。当然，大多数人都是报喜不报忧，当中有多少真实成分我母鸡，但不管怎么样，她的确很牛逼，在理财方面。在做人和当员工方面，也的确很狗屎就是了。

那。。这么说起来，我的标题就不大严谨了，狗屎只是她的一部分。我应该改成什么呢？狗屎又牛逼的 D？不对，牛逼好像也很宽泛，狗屎，理财又很牛逼的 D？这样好像更严谨点。但狗屎的 AOE 范围是不是太广了。Emmm，严谨真难啊。还是狗屎吧，因为我打算直接管自己的文稿叫做狗屎文稿，这样就不需要理会狗屎的严谨了。对不起，严谨被我蒙上了狗屎的污名 Orz，哦，我也是个狗屎的人。

劈个叉，我到底从哪里学来的狗屎。都说了一个星期了 @。@ 想不起来了，就像我想不起来周四在候车的时候，脑子里闪过的脸，她似乎是我之前看极限挑战里的群演，但无从查证。就像，我不知道我从哪里学的狗屎一样。
