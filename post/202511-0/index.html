借贷吗？ @11.01

这是前天的事情，但又懒得记在那天，就当作今天的日记主题了。那也是面完 Dify 的那天，下午 Jerry 找我帮忙发一下他的招聘信息，他的一个固有印象就是我有很多前端工程师的微信好友。嗯，没错，毕竟掘金也要，思否也好，用户会添加运营人员的微信好友，而这波用户前端的占比估计能到七成，到八成。但，在 nebula 呆了四年半，这群人就是个微信好友而已，真要我找谁，问他要不要换份工，竟然没有一个名单。所以，就发了一条朋友圈。

可能是我炸尸的缘故，nebula 的前同事们开始小窗戳我，一二三四五，同步完彼此的近况，就迎来了冬日般的沉寂。接下来就是找研发们看 PR，别说我不在其位，即便我在其位，总有人会抒发不满，抒发完 PR 也不会处理，严格来说就是单纯的抱怨吧，这些 PR 不行，我没空，无啦无啦。横竖，我能做的就是找人看 PR，效果，真难保障。因为闲聊，有两个前同事开始说，你存款是不是不大行了，我可以借你点。我回绝了，虽然很心动。但我知道这份心动是一种类似死缓的拖延感，因为有后备，有人可以借我钱，所以我就那么想去找工作。断了后路，好好找工作，对我而言才是最好的。

不借贷，不找人借贷，无论是我借钱给别人，还是别人借钱给我，结果都只有一个，大概率是失去彼此。借钱给她/他的，我会嫌弃她/他不守约定，一开始说每个月还 1k，后面就 500，再后面连借口都不找了，直接推翻了先前的月还一千的约定，以 500 的标准执行着。那借出去的一万，我用了一年半才收回来。还有我哥的，七年没提过还钱的事。当然，有好的体验，小俊俊的欠款是按时还我的，有时候还会比约定的 1k 多还 1k，甚至。。我都忘记了借了他多少钱，横竖他会记着。顺顺利利提前还完了。我也借过钱，租这套房子的时候，找 mona 借的钱，因为要年付，那时候我只有半年的房租的存款，欠款三个月还清了，但考虑到利息，我也不知道咋算，最后送了一个索尼的头戴耳机给她。无论是借钱，还是被借钱，和人打交道，都不好。即便我的存款不行了，我也会找银行借。银行会收利息，不带感情的两清是最棒的。

当然，有人愿意借钱给我，我还挺开心的。就和当初我知道别人收入困难，问她/他要不要借钱给她/他一样。

昨天回顾十月读的书籍的时候，没算上的《零售的哲学》，711 的创始人的 711 发展史记录。也是今天打开这台电脑的浏览器，发现窗口有一个阅读记录是这本书，才想起来，哦，我读完了这本书。翻看微信读书的“读完”记录，没有其他的书籍了，无啦无啦。

十一月继续，好好读书吧。目标是吧三本 Python 数据分析的书都读完。

享受咖啡

除了来同步近况的 nebula 同事之外，其实那天慧慧同学也戳了戳我，我还以为她看见了我发的朋友圈，所以来找我聊天。结果一对暗号，发现她只是单纯今天想起来了我，而刚好今天我又发了一条她没看见的朋友圈。忘记扯了些什么，说到了咖啡，我说我能喝的咖啡分量是 1/2 杯，大概是 120-150 ml 左右。她说，是心悸吗？我说，不是，只是过了这个量之后，我就不是在享受咖啡了，剩下的 120-150 ml 是冲着不浪费喝完的，虽然喝完也不什么不适，但没有太大的愉悦感。她说，她每天喝两杯，上午一杯，下午一杯，不是一次性泡两杯。我说，本身冲煮过程也是享受咖啡的前奏。

今天打开了两周前买的 blue bottle 的东京拼配，不知道是别人的退货还是补货了，关注这个新品一段时间了，终于在降价且有货的时候下单了。今天打开也不是一时兴起，而是路过放着咖啡豆的岛台，发现这包没拆封的豆子异常鼓，觉得奇怪，毕竟我应该是封住了出气口，里面不应该会有过多的气体交换，结果一看。。。我压根没对这包豆做保鲜处理。好奇冲了七天的气，这包豆子啥味道。打开，倒出来，发现颗颗豆子油光发亮的，看豆子的颜色，应该是深烘豆，所以先试试用 9bar 去压一压，看煮出来啥味道。结果，咖啡液出得速度贼慢，力量的出液速度是 25-35 秒出 35-45 克的咖啡液，这次过了两分钟了，出液体量不到 10ml。原以为是管道堵塞了，拆了，清了，其实没清理出什么，管道是畅通的。又煮了一壶，还是比原先出液快了一丢丢，也只是一点。还是很慢，两分钟也没出完。

拿出被收起来的另外个 9barista，想煮一壶试试。结果发现，它出液也很慢，不至于两个壶同时间出问题。去小红书搜了搜，出液慢的原因。发现一个偏好出液慢的咖啡爱好者，为了出液慢调细了咖啡研磨度。但，我的磨豆机，没有调整过研磨度，理论上不会有颗粒度偏差，而且上上上次出液很顺利，也是这个颗粒度。但，本着实验求证的原则，还是调粗了一个颗粒度。这回出液正常了，相对顺利地出液完了。但，应该要再调粗一个颗粒度。所以，磨豆机出啥问题了？估计得明天拆开看看了。

喝了一口东京喫茶拼配，很有去年在北海道买的豆子的风味，深沉的苦。好苦。

白粥开花 @11.02

连续点了苏小柳三天的海鲜泡饭，在第三天发现虾一点都不紧致，蛤肉少得一比之后，打算自己在家熬粥喝。明明是一个 5kg 大米，去年十一月买来之后，到现在还没吃完的人，却分外喜欢熬粥。慢慢煮粥的过程，和之前炖牛肉汤一样，虽然耗时比平常做饭要长，却不排斥这个慢慢制造食物的过程。尤其是，看着大米泡了之后，微微膨胀，泡在水里慢慢呼吸，冒出的白色泡沫聚集在一块，白花花的一片，这是米饭才有的生气。

昨天喝的是蔬菜海鲜粥，今天则是玉米牛肉粥，在盒马买的玉米、豌豆、胡萝卜粒半成品，等放进粥里，才发现自己放少了，只能下次再改进了。怎么都煎不好的汉堡牛肉，这回直接捣成碎片，加了酱油和油，腌制着，等快盛出的时候，加进去。一起在等候区的还有蒸熟的杏鲍菇，是为了提升口感来的。为了不至于大材小用，这回蹭了番茄的小蒸锅，巴掌大的蒸锅，平时给番茄蒸三文鱼和牛肉用的，这次用来蒸杏鲍菇刚刚好。先把杏鲍菇放下去，撒盐，加上不要钱的白胡椒，难得白胡椒有发挥之地。最后是牛肉，汉堡肉的牛肉太碎了，碎到放下去，搅散，混均匀之后压根不见踪影。

出锅，出锅，等着粥放凉的时候，把今天的日记写了。

小花狗不见了

这好像是「铁齿铜牙纪晓岚」里面的一句台词：“天塌了，地陷了，小花狗，不见了”，大事不好了。用这个来形容最近不大妙的记性，再适合不过。上次面试，说起大部分程序员有一种 乐于分享 的天性，面试官说起了《大教堂与集市》。我说我看过这本书，但忘了。不止这本书，很多书我看了，都忘了。十月看过的《零售的哲学》，甚至我都忘记看过了这本书。但，现在记得名字，我也记得它讲了什么。再往前面《精益数据分析》里面具体的六个产品形态的数据指标忘记了，再往前，去年读过的一系列伊藤比吕美的随笔，忘记的差不多了。

记性变差，似乎和年纪有关？撇开年龄原因，和我如何读这本书也有关系。我读书的时候，到底在读什么？提问了吗？回顾了吗？有一点，不多。还有就是睡眠问题，最近的作息，七点半要起来拌饭，晚上又做不到早睡，睡眠时间不到七小时，困得一比，脑子估计都没好好休息。早睡，从今天开始就早睡。

女性的哭声

昨天下午五点左右，外面突然传来了悲伤的哭声，是一名不知年纪的女士的哭声，很悲伤，嗷嗷的哭着。我听着声音有点不知所措，而这个不知所措是想做些什么，但又不知道该做些什么。光听声音，我无法判断是左边的邻居，还是右边的邻居，可能还是楼下的邻居。哭声持续了五分钟左右，安静了下来，似乎一切都没发生。

所以，她到底遇到了什么事呢？如此悲伤地哭泣呢？这是她人的故事，我只是一个听众。望着如此明媚的冬日，希望她的心情也能明亮起来，温暖起来吧。这是作为一个听众，能有的期盼。

负反馈有用吗 @11.03

从十一月份开始，给每天的日记后面加上了日期，是为了不知名谁的读者加的，经过近五个月的更新，这个博客的每周访问人数从 6 个增加到了稳定的 12-15 个，当然我的 IP 还是占大半，这回不只是有德国的我，日本的我，还有 Estonia（爱沙尼亚）的我，澳大利亚的我，台湾的我。我的梯子近三个月异常不稳定，Estonia 节点是最稳定的，虽然延时很高，但能用，其他节点红成一片，和 Shadowsocks 反馈了，工单来回回复了四轮，丝毫没有解决的迹象。总之，为了 12-15 人次中过的一半非我的访客，章节添加上了日期，万一有人在追更呢 —> —>

这个话题其实是昨天想写的，但。。昨天也说了，记性开始不好了，所以选题忘记了。想起来又忘了，努力回想，终于把这个话题记住了。上周在小红书看到了综艺「再见爱人第五季」的讨论，于是也入了坑，拿着淘宝 88vip 换了芒果TV 的会员。从第二集开始，因为某位嘉宾涉及贿赂敏感（没去追踪事实如何），所以这组夫妻（嘉宾和其妻子）的镜头就取消了，然后小红书吐槽一片。有意思的是，我开始关注事件发生之后，这档节目的观看数的变化。一般来说，视频刚发布的时候，增速是最快的，观看人数会笔直上升，然后慢慢减缓。这是一个趋势，几乎所有内容都符合这个规律，唯一不同的就是增速放缓之后曲线的变化，这是后话。第一集的收视是 7kw 的观看人数，第二集是 6kw，第三集是 5.5kw，其实没差，因为第一集有时间加成。我开始思考，即便观众的反馈都是负面的，吐槽的，不爽的，但她/他的观看需求不会因为内容的丰富性下降而直接取消，看，和不看，观众只有两种选择。虽然我不满你的内容删减，没啥意思，但剩余两组嘉宾我也想看，即便我最想看的一组嘉宾没了。就结果而言，观众还是会看，只不过评价是负的。负反馈，似乎一点都不影响结果。

同样的，负反馈不影响结果的，还有山姆。换高层、换选品、改用户条款…，用户都在吐槽，但就结果而言——因为没有替代品，用户还是会继续使用。按照积极、阳光来说，两个事件都不是什么好现象。但，这种现象会越来越常见吧，归根到底，是一种资源的垄断？假如有替代品，估计事情就变得不一样了。

hot and cold

平时都是看 Reddit 的订阅推广邮件，难得上一次主站，在侧边栏发现了小游戏入口。一开始是一款叫 hot and cold 的游戏。看到第一眼，不知为何脑子想到了数据库的冷热数据。（😂）想p 呢。

这是一款猜词游戏，谜底是一个单词，你可以尝试猜测一个词语，例如 pizza 之类的，系统会给出这个单词同谜底的靠近程度。蓝色的排名结果说明这个词和谜底相差很远，排名的显示颜色越向红色靠近，说明你越接近答案。从大概 300 名开始，排名会有太阳图标 🌞，字色是淡黄色，说明你快接近答案了。再往前可能是 200 名，是火的图标 🔥，字色是橙色，前 100 名的排名字色已经接近橙红色了。不只有图标、字色，还有排名来说明你的接近程度，有 39994，五位数的排名，说明几乎没关系，排名 1 的话说明换个说法就好了。比如，有一次谜底是 child，我猜了 kid 一样。

这个游戏应该迭代过几个版本，不知道几个，我玩过两个版本，上面说的有排名数和图标、字色变化的是我玩的第一个版本，还有一个显示百分比的，越靠近百分比越高，字色也是从蓝色调往红色调走。但那个谜底我没猜出来，最靠近的是 77% 的 water，其次是 pond 和 pool 都是 74% 的近似度，sink 和 basin 是 71%。

大概玩了五局游戏，最后一个谜底是 fish，我先猜测了 cat，排名 510，然后我猜测 dog，排名 283，meat 有 228 的排名。就往肉类猜测。最后凭借别人的 hint jew，我输入了 fish，答对了。。就是这个 fish 让我放弃了游戏，fish 和 cat 的关系竟然还不如狗？？？这个游戏不要紧吧。

这算是一个负反馈对最终结果生效的例子，至少昨晚的我还很沉迷这个猜词游戏，今天的我已经丝毫不想玩了。

外出失败 @11.04

昨天打算着今天起来放饭之后去跑步，昨晚的睡眠还行，中间虽然醒来了半个多小时但至少不是两个小时的混沌醒。连三星健康都说，昨晚睡眠 90 分喔。但还是困，也许从一开始我打开窗帘发现十一月七点四十的清晨如此亮堂，完全没有黑夜遮我影的安全之后，就打消了外出跑步的念头。

自然，跑步还是要跑的，今晚继续早点睡，比昨天早半小时，看明天状况如何。跑步啊，怪不好意思的，请教了 yihong 不少事情，却到现在就跑过一次。以隐形眼镜为分类的话，别人是日跑（抛），我是月跑啊。

昨天看 Python 数据分析的时候，想起了上周四和小乔治的对话，你最近干嘛呢？学 Python 数据分析呢。学那玩意干嘛，现在我用 ai 写代码又快又好。（找托辞）学原理呢。对话结束，其实我并没有学什么原理，我把一个个例子都看进去了，记不记住就另说了。学原理变成了一种美化说法，遮掩我在学 Python 的事情。似乎到了现在阶段，说我在学语言，别人会觉得你嘛呢？ai 那么便利。也许是一种无用功的变种吧。也许我说自己穷逼，没钱雇佣 ai 写代码会相对真实点，当然这也是客观事实。但归根结底，还是一种我在学语言和时代不相称的违和感引起的“找借口”。要说我喜欢敲字符，看它们如期运行很快乐，自己写的代码会带来愉悦。这似乎会引出可控的信任危机。自己写的东西更加了解它，更可控呗。谁知道，反正谁来问，就以学原理糊弄过去吧。说回 Python 数据分析的事，看书的时候，发现无论作者多么体贴读者水平，都无法方方面面照顾到你的水平。于是，我让优化 prompt 的 Gem 给我优化了一个 Python 数据分析指导的 prompt，给我讲解相关概念。这不，我知道了复数，以及它的使用场景。

迭代，这是第几代

推上最近有一股“先上/先做再说”的激励人迈出去的说法，不禁地将它和“草台班子”联系在了一块。后者是大前年开始流行的说法，是一种我从未采用过、却看过无数次的说法。我觉得它是一种不负责任，不改进现状，以单一元素描述事物的方式。因为草台班子啊，暗含着“就这样吧”，“能怎麽样呢，别人太菜/水”的想法。不好，一点都不利于改变现状。但说此话的人不负责任的我，也有点不负责任。评论事物涉及的团伙，或某个人为草台班子，也许这个人和事物并无关联。再者，即便有关联，让这个人不要单一化归因，抒发情感不满，也有点过于苛刻。呸，抱怨，吐槽都是舒缓情绪的方式啊。说回草台班子和先上理论，我萌生了一种二者是因果关系的想法：因为先上，自然不会多完善，就导致了草台班子的评价。

但，我没将迭代引入这段关系中。其实，先上一般后面会跟着再迭代。自然，迭代跟没跟上另说。就固定模式上，迭代是在先上后出场的。不知为何，我说起迭代，脑中总会想到程序似乎也有一种类似机制。哦！那不是迭代，它的名字叫递归。虽然二者都有一次次执行的意思，但不一样不一样。

可能是因为天气也知道我不爱运动，这周一直在下雨，周四的雨直接把我封在了家里。周五的雨倒是小了一点，可以外出丢个垃圾。今天，也是微微的小雨。看了下天气，明天有概率降温。但，这周只剩下明天了，所以今天要十一点睡，这样明天起来给番茄和葱白拌饭之后，我就能外出跑步了。这回，放在冰箱里呆了一周的猕猴桃可以出场了，作为运动前的补钾食物。明天一定要跑步成功。不知道哪些每天都跑步的人，是如何解决下雨问题的。难道是雨披，雨披我倒是没有，真哪一天我穿着雨披去跑步了，估计跑步已经变成了我的生活必需品，是一件认真对待的事情了。

跑上了，在泡了 @11.09

原本早上起来拌完饭，想着回被窝继续睡。但想到今天开始改小猫咪的饮食计划，九点那顿本来由自动喂食机放干粮的餐，改为了继续拌湿粮。这也是为了后面工作了，调整的饮食安排，确保番茄每天能摄入 180ml 的水。继续九点要去拌饭，那现在滚回去睡也只能睡一个小时，而且上去之后不一定立马睡。不如索性出去跑步吧。忘记给自己留一壶净化水刷牙，所以去厨房水龙头那接了一壶由滤芯水龙头过滤的水。刷牙，洗漱，喝水，吃猕猴桃，和小猫咪们道别，出门。

出了楼栋，习惯性往左走，突然想起来这个点，西门在运输垃圾应该开着。扭头，朝着西门走去。出小区门，步行，等红绿灯。过马路，结束佳明的睡眠模式，开启跑步模式，do workout？是什么。不管，先把跑步界面调出来再说。按下开始计时，开跑，跑 60s » 走 60s » 跑 60s » 走 96s（遇到了一个红绿灯）» 跑 96s » 走 60s » 跑 80s（到小区了）» 走到家。就这样，第二次跑步结束了，跑走的模式是 Gemini 为不怎么运动的我定制的，下一个阶段就是跑 120s，走 120s。。

最近的天气，加上配合番茄和葱白的饮食作息，应该能每天早上出门去跑一跑。回到家，冲洗了下浴缸塞子，放下塞子，放热水。打开去年买来没开过的薰衣草味浴盐，倒进水里。关洗手间门，和小猫咪们打招呼，挪走趴在椅子上的葱白，开始写今天的日记。

今天跑步的时候，可能是不怎么运动，最近吃的也不多。感觉脸上凉凉的，我看不见自己的面色，但似乎有之前犯低血糖的感觉。大概率是低血糖了，看样子得好好的吃饭，作息才行。泡澡的水还在放着，小猫咪的第二顿早饭还没到点，刷刷推，看见珠海男子手持菜刀随机乱砍的事情。啊，我不懂这些轻贱他人生命的人，为何如此看重自己的生命。真不想活了，自己抹刀便是。当然，这个是暴言，估计怎么样都想活吧，但别人也是啊。

最高の木鱼花

中午用快开的水焯了 10s 的鱿鱼——为了炒锅炒鱿鱼不要留下白白的底——过冷水，和蛤肉一起丢进热锅里炒，炒了快一分钟，把芹菜、洋葱、请教丢下去，开始搅，搅，搅。不得不说，一人锅炒这种配菜多的菜，显得有些小，把底部的鱿鱼和蛤肉捞上来，费了老大劲。洋葱变透明了，再搅一搅，捞出来盛盘。加上另外一道盐撒多了的清炒青菜，连蒜片都没放，其实挺好吃的就是咸。作为零嘴，吃了两个冰箱里拿出来的大豆腐泡。

吃完过了两个小时，人就开始不对劲了，不知道是放了十天的洋葱在冷藏室的诅咒生效了呢？还是泡了一个多小时的鱿鱼有太多不甘，或者是只被我吃了两颗的豆腐泡早就升天。总之，这些食物连带着早上十一点半喝完的咖啡，开始让我的胃很不舒服，俗称的吐了。一般做饭，菜量分为两餐。看胃这个蠢蠢欲动的架势，午饭吃的东西是不能再吃了。所以打算煮个超级清淡的泡面。把冰箱里的奄奄一息的西兰花拿出来，用剪刀弄成大小均匀的小颗西兰花，撒了一把盐，抓了十几下，用流动的水清洗了四、五遍。用水泡着，泡了一个小时的样子。煮泡面的水快开了，去捞西兰花的时候，发现焉巴巴的西兰花，变得很有元气。像是一周之前我刚买它的模样，西兰花梗是充满水分的坚挺。煎了一颗鸡蛋，两边起了脆皮之后，就丢去煮泡面的水里煮着，捞出来的西兰花丢去煎鸡蛋的平底锅里小火煎着，旁边是正在煮泡面的小锅。

似乎忘记了木鱼花是如何出场的。简而言之就是在煮泡面的水加热了 30s 突然想起来晚上是打算把木鱼花泡水，过滤掉木鱼花之后，再将其丢去电陶炉煮开的。所以，从冰箱拿了一把木鱼花在水里泡了两三分钟，在这个空档，在另外一个电陶炉煎鸡蛋。

这个泡面来来回回一共花了二十分钟，然后屋子里充满了木鱼花的气味。西兰花最高！

盲了 @11.10

「失明漫游记」看完了，有些怅然若失。一部分是来自于微信读书显示这本书有 700 页，但我读到 645 页就结束了，后面是译者说和作者随笔。进度条会有一种靠近结局的预判，我以为还有一个章节（这本小说一个章节大概 50 页）才结束，后面的情节应该是不是这样发展，想着这个，突然就完结了。有一种仓促的散场感，像是看一场以为两小时的电影，却在一个半小时，片尾字幕出来了，电影院的大灯亮起来，人群们开始往出口走。

该说我读的小说少呢？还是读的好小说少呢？这本书，在我读完三分之一，近二分之一的时候，已经被我预判为今年的年度最佳小说了。可能是这本书有一种真实感，读别的小说，无论是纪实性质的「黄金时代」、「流俗地」，还是基于现实的虚构小说「将死未死的青」、「GOTH 断掌事件」，都不会过于代入。但「失明漫游记」的设定是，这是一场人们不知如何被感染的传染病白色眼疾在人群中蔓延开来的故事。先是一个人突然失明了，然后去就诊，看诊的医生随后也失明了…如果我失明了会如何，这个想法在我大学的时候突然冒出之后就扎根了，偶尔自己会闭上眼睛去感受下周遭事物。比如，洗澡的时候，闭上眼睛去摸索放在高处的洗发水，一旦能摸对，不是摸成了沐浴露就会很开心。与其说是感受盲人看不见的体验，不如说在给自己生活添加调剂。但，这本书不是调剂品，如果全人类都失明了，那理性和文明该如何存活，“人性”如何体现。在大家都看不见，不知道随地大小便的人是谁的情况下，追责变得困难，负责似乎就更加可贵。

（剧透）结局也很有意思，我原以为唯一看得见的医生夫人到最后是要失明的，但也许她的确失明了。其他是陆陆续续复明了，复明的过程很有意思，就像在祈祷“我要睁开双眼看看这个世界”，一睁眼就真的看见了。

记得小时候，有本名著，叫做「假如给我三天光明」，十分有名，有名到我让我妈买，她都会不问缘由地掏钱。当然是她想让我多读书呢，还是真的读过这本书呢，就不得而知了。到这把年纪，这本书我还是没看过。反倒读完了它的相反书籍“关于全人类失明”的这件事。

偶遇一只晒太阳的橘猫

去驿站拿包裹的时候，刚出楼栋大门，就看见在高台上，有一只橘猫以母鸡蹲的姿势在晒太阳，它和我在小区见到的其他人喂养大的流浪猫不大一样。我距离它两米远，见不到它清洁情况，是否长虱子。但就两米外看来，这只猫的毛色很润，有光泽，应该皮肤状况也不差。最重要的是，它让我想起了番茄。。因为它和番茄的体型很相似，鼓鼓的肚子，但不是猫咪传染病猫传腹，因为它的脖子也很圆润。总之，是一只和番茄一样，胖得很均匀的猫。它应该是走失了，前饲养员养得很好。想起八月底的时候，在楼栋电梯口见过寻猫启示。依稀是一只橘猫，好像是这只。

但我想到这茬的时候，突然有些遗憾，我应该留下来饲养员的联系方式的。也许，下次我看见寻猫启示，可以拍下来。这就意味着，以后我外出就得带手机。也不是不可以，穿个有兜兜的裤子就好了。

十一月上旬就要那么结束了，最后说一句，今天的牛肉粥超好吃！！

黑底白字读着字似乎不够清晰

  • 现象：电脑开的暗黑模式，draw.io 自动读取了系统的设置，所以画布也是黑色的，发现黑底白字的阅读效果不如白底黑字。
  • 原因：光晕效应，瞳孔在“看”明亮的白色时，会缩小，自动聚焦，会让文字更加突出；反之，黑暗的黑底时，瞳孔会放大以便接收更多的光线，聚焦会更难。所以，暗黑模式会更护眼，是因为眼球不是长期处于紧张的聚焦状态。而光晕效应发生在有散光的人身上，白色的光线会“渗入”到周遭的黑色背景中，让文字边缘变得模糊不清，降低了黑白的对比度。——嗯，我有散光。
