这条拷贝的是四月的上条文稿，它说我续费了 midjourney，想到两个星期之前 mj 的创始人说 Google 的 TGU 好像是这个，也许是 TUG 导致了 mj 的训练慢了一整年。说实话，我其实没觉得 mj 比哪个慢。我依旧很喜欢这个姑且被我称之为艺术的产品，同类产品的即梦，也有很多创作者和设计师在用，诶，emmmm，设计师也是创作者啦。但，我总觉得 mj 更富有艺术气息。

code review 和修改稿件

昨天要给 binbinzai 修改稿件，咋说呢，我第一次体会到了 cr 的心情。我好像给他重新写一遍，但觉得这样好打击他，所以作罢了。就改了有点点友好度。

突然意识到，稿件和代码一样，我不知道写代码的人追求啥，可能是简洁、可读。后者是我的追求，我觉得一篇文章可读性是一定要保障的，所以我十分想给那篇稿件。但…还是算了。

我…又一次在剖析自己的心态，我为什么会放弃质量，选择去考虑他人的心情。这是不是不够敬业呢？我不知道。大概，我潜意识里觉得自己的标准其实也不是很专业的判断吧，既然我的做法不专业，那别人的做法有什么好修改的呢？也许，试试这样也行。

宝宝和狗屎

由于觉得狗屎这个词，过于粗鄙。从这周开始，我用宝宝来代替了这个词。小组来了个实习生，为了怕她不懂我在说啥，我和她说，你自动把宝宝代入狗屎就好了。

于是，她日常见到了我在喊，这个视频真宝宝…这个网络宝宝极了…，今天啥都没干，又产出了一堆宝宝…小组充满了童真，全是我发的宝宝 Orz不知道这是加重了我的粗口呢，还是真的起到了约束作用呢。谁知道呢？

chromebook 和 chatgpt

周六羽毛熊说想我给他推广知识星球，当然在此之前，我已经和他说了，我的推广能力就是宝宝。这句话，我在周会也说过…当然这会没有说宝宝，说的是…我的推广能力就是狗屎 Orz

然后他说在用 deepseek，我问他要不要出来找份工干干，人会更有精神？不知道每天看着一些动漫（我之前是看剧集），一天就那么过去了。所以我想着，我入职之后，用上 gpt 和即梦之后，有了一种重新连接上社会的感觉，我想也送他一份年度的 chatgpt 的会员，至少他能用 codex 写点代码，比不上 claude 至少不算太脱节。

于是我开始研究如何开通第三个 gpt 会员，我两台手机，索尼的绑定了给公司打工的主账号，全是我对公众号文风的要求：自然且克制；pixel 2xl 给我拿去给自己的日常个人用邮箱开通了 gpt 会员，拿来当备份，以防止我公司账号生图功能到了上限。说到这，五月我达到了上限，开通了 pro。后来我觉得，2 个 plus 够我用了，pro 可是 10x 的 plus。我真是败家。

回来，我就开始研究如何用 pixel 2xl 开第三个会员。我好不容易搞定了它的网络，clashx 突然把我所有的飞机场的飞机都下掉了，订阅的所有飞机都没信息了，给了我一个提醒请用支持 trojan 的客户端 Orz 好不容搞定了，发现无论我如何退出，google play 默认新账号，都不能在 pixel 2xl 开通第三个账号，老提醒我已经开通了订阅，点击已订阅，跳转是没有订阅的空页面。大概率是手机绑定了订阅，一台手机只能用给一个账号订阅。

于是我想到了，是不是用安卓模拟器。但是似乎 Google Pay 有严格的环境监测机制。于是，我想试着用 apple store，若干年前南浦月送了我一个美区账号，但这个美区苹果账号绑不上我的 master。Orz，于是我咸鱼 200 淘了一台 pixel 3。

这算是我 2xl 之外，比较喜欢的一款 pixel 手机。在淘手机的过程中，我甚至想，要不换个手机，直接把索尼换成 pixel 10，但小索尼还能用呢。最终是花了 200，这台我打算拿来绑定第三个 gpt 订阅的小手机，明天就要到了，希望我能顺利地送一个 codex 给羽毛熊。

说到这，在安卓模拟器放弃，apple store 失败之后，我想到了既然都是客户端，是不是 chrome book 也可以。我拿出来了放抽屉里许久的 chrome book，没能升级系统，Google 官方说已经不在维护了。chrome 没法升级，chatgpt 的页面是样式丢失的，Google play 是找不到 chatgpt 的，有意思的是 YouTube、Spotify，都支持，也是，这都好多年的应用了。所以 chrome book 就那么被我充满了电，放着了。pixel book，这是这台 chrome book 的名字，Google 官方出品的电脑。其实很好看，它充电的进度条，非常漂亮，是个四色灯带。但上次充电，好像是五年前？或者是六年前。不得不说这些设备，真的好强，过了那么多年，还能充进去电，继续被使用。虽然屏幕有些漏液了，但基本上没有其他损坏。

空虚和孤单

今天和黑哥哥摸鱼聊天，我说我周一是两点睡的，不是加班，就是陷入一种空虚，不知道自己想干嘛，就是不想睡。在一张德州桌子上，一局接着一局地打着德州，不知道自己在等什么。既不是筹码，也不是完成任务，就那么不想睡觉地打着德州。

黑哥哥说，这听着很悲伤。感觉有点……后来，我才知道，原来那是孤单。就那么无所事事地，空虚地度过着时光。我原以为是自己没有什么想做的事情，过于没有成就感，所以 emo 地在玩德州。事实上，被排期追了一个半月，我心情不怎么好。工作日为了排期能正常发布，就加班。整个人状态都不是很好。周末空下来了，就感觉人生没有意义，一局接着一局地玩德州。

当然，还是没充钱。原来这是孤独啊。行吧。还是给自己找点事情干干吧。
