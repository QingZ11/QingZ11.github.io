GitHub Pro trial

突然想给这个博客增加一个 tab 用来全部展示图片，又一次启动了 Copilot。Claude 干了没多久，就感觉默认的 Sonnet 3.7 不大行，想换个模型，发现我的 free 版本只有那么几个模型，那怎么办？升级呗，刚好有 30 天免费试用。就开始了试用之旅，用着 Sonnet 4.5，其他模型都要填 API key，无啦无啦，用 4.5 吧。毕竟我的需求也很简单，Sonnet 干了两个小时不到就完了，当中还包括验收时间。

因为是图片，等所有需求实现已经十一点了，图片压缩的网站传图速度也很慢，没啥时间搞图片压缩，先上了。等着早上给番茄和葱白放饭了，就开始压缩图片，以及优化图片加载方式。昨天实现的是一次性加载，好卡，当然图片也大，都是原图。和 Sonnet 4.5 说上懒加载吧。一次性加载 27 条数据，就是 9 行数据，因为我不喜欢 10。这个需求一次性过了，现在就等着 300 多张图片压缩完成了。等以后更新，再慢慢把其他图片补上。

图片现在排序方式是时间降序，对观看图片的人来说应该是友好的，但就我而言，似乎过去的时光浏览着，更让我珍惜现在的番茄和葱白。所以，哪种排序好呢？

昨天开始读一读 saka 在推上分享的 Uber 从 PostgreSQL 转向 MySQL 的文章，我不知道这文章算长还是短，读了我 2 个小时，才看了 50%。先看原文，再看刷到的宝玉的翻译 prompt 的翻译结果。对着原文和翻译再看一次，再查个别不认识的词汇。就这样，时间过去了。

多次提交

Claude 的二次打工应该结束了，在博客功能性修改和内容性维护相互交杂的时候，commit 的信息就多了一种选择，是分开提交还是一次性提交。偷懒的我，选择了一次性提交。后来想想功能性修改应该和日常性的内容更新区分开，所以在最后一次功能性修改完成之后，提交了本次的图片排序修改。在最后一次，终于算是清理干净了点 commit。

天气冷了，番茄和葱白都开始往高处走了，即便是挨着我就近窝着，也选择跳上我的椅子，而不是在地面趴着，所以日常就变成了一人两毛三椅，自然一排是放不下的，构成了一个稳定的三角形，番茄坐着平时我坐的转椅，我和葱白坐着折叠椅，葱白坐新买的白椅子，我坐之前书桌的那张黑折叠椅。彼此就在触手可及的地方，当然是人类的手，而不是番茄和葱白的小圆手。它俩就趴在椅子上睡觉，我就看电脑。画面很和谐，拍不进去我的小相机。

昨晚睡着睡着，感觉番茄从椅子上跳上来了床，靠着我。因为腰又开始疼了，被番茄压着的被子告诉我，这货又要和我抢被子了。我也不知道他靠着我睡觉舒服不舒服，我被被子的压力扯疼的肌肉告诉我这不舒服。没多久，他就靠着我开始用后腿蹬脸，迷迷糊糊地想着，这货还挺厉害，隔着那么一个大项圈，还能蹬到脸呢？就那么睡过去了，结果他越蹬越厉害，腰背传来的蹬腿节奏越来越频繁，伸手一摸，哦，他把项圈搞掉了，不愧是番茄，总有法子弄掉让他不适的东西。又那么睡过去了，早上起来，看见地上一撮被染红的橙色猫毛，想来番茄的耳朵又被蹬出血了。本想作罢，等着明天伊丽莎白圈来了之后再说。看了一眼他耳朵，还是把原来的小鸭子项圈戴了回去，好歹能支撑一天吧，虽然早晚这货会想着把它摘掉。番茄的恶循环：耳朵有马拉色菌，痒»不给洗耳朵»痒，就蹬耳朵»蹬耳朵蹬出血»戴项圈»洗耳朵，逃跑»耳朵快康复了，摘掉项圈»因为戴项圈不舒服，所以蹬耳朵»蹬耳朵出血»蹬耳朵蹬出血»戴项圈»耳朵快康复了，摘掉项圈»因为戴项圈不舒服，所以蹬耳朵»…只能叹一口气，就这样吧。

狗吠和蝉鸣

杭州一下子就入冬了，看着衣柜里的秋装，不得不说这是一笔失败的投资，杭州是没有秋天的。至少今年如此，往年也如此，不过往年留了一个、半个月的时间让我的秋装出个场。开始在家穿上厚厚的袜子，也不外出就是为了暖和一点，九月底就在阳台晒着的被子终于被装进了被单里，那时候的天气预报告诉我，十月开始就一直下雨，结果，雨没来，降温也没来，拿出来的被子就这样在阳台放了一个月，大概积攒了不少的灰尘，又被装进了袋子里，如果被单是袋子的话。下完雨就带来了冬天，一下子蝉就不见了，听不到半声虫鸣。

也许是为了不缺这部分的声音确实，楼下的楼下，也许是楼下的楼下的楼下，隔壁的隔壁，隔壁的隔壁的隔壁，简而言之楼下某层的某户，忽然间多了一只狗，也许是两只，也许是三四五六只。但有一只狗，在努力地补位夏蝉。我不知道它什么时候入住的，是搬来了一户原先就养狗的人家呢，还是原先的人家开始养狗了呢？原因不得而知，结果就是不知道什么时候开始一直能听到小狗的叫声，它应该是一只体型不大的狗，体型大的狗的声音更加低沉。而这只小狗也许是寂寞了，似乎从我起床到睡觉，都不停歇地叫着。所以，它的主人早出晚归，不怎么在家吗？反观我家的番茄和葱白，我在的时候叫唤也挺频繁的，不过它俩讨食的声音穿透力不如小狗的寂寞，也不如小狗的坚持。所以没有什么邻居投诉，当然小狗似乎也没什么邻居投诉，至少从声量和频率上来看，持续着一种稳定态。据说，猫的叫唤是为了和人类“沟通”，当猫只剩下猫或者空气时，是不会有“喵喵”声的。

叫声，真是一件有意思的事情。比如让人惊醒的尖叫，让人感动的雀跃，二者都着高分贝，但前者让人害怕，后者让人开心。

程序员节快乐

若干年前，依云吐槽过，真要过程序员节，应该是 913，而不是 1024，前者是每年能过上的最大 2 进制数 256 的天数，一个人的声音怎么能盖过一群人呢？所以 1024 继续着，913 页就被我记住了而已。所以，程序员节快乐 ☕️ 在思否开始接触这个群体，就形成了一个固定搭配：代码 + 咖啡 + 睡觉 = 程序员，似乎是一件 T恤 上的 slogan，或者说是某个 slogan 被印制成了 T恤。而我的咖啡启蒙，不算雀巢咖啡粉的话，应该就是思否的栋哥教的。SF 有一个滴滤咖啡机，栋哥和慧慧同学负责买咖啡，我蹭了一杯，后面也加入了买咖啡行列。买的是 illy 咖啡粉，滤纸不知道，现在想来应该是 Hario，因为从小喜欢中药——惭愧的是现在是个中药无脑黑——咖啡的苦味和烟草味，和中药如此的相似，自然就喜欢上了这种饮品。可惜，喝过一两次之后就开始心悸，后面就蹭栋哥和慧慧同学煮咖啡的香气，想来 SF 的工作还是很开心的，因为写 PHP 的栋哥会讲烂梗，写 JS 的张祖（事实上他本名不叫这个）会抖机灵，冷不丁抛出一个包袱的干云，一本正经的三三，他们都是程序员，午饭时间就听他们讲笑话，那时候和另外一个运营 lirong 说，我其实挺喜欢去办公室的，因为一个个有趣的灵魂在空中飘着。她说，那多恐怖啊。怎么会恐怖呢？一旦和他们肉体接触，灵魂就落地了啊。

再后来就是 SF 工作认识的公子、依云、羽毛熊，虽说那时候自己不年轻，但还是要感叹下年轻的时候遇到的人真是有趣啊。再后来掘金遇到的 Next，阿德，王总，也是非常有意思的人，也都是程序员，分别是安卓开发、Node.js 开发和 PHP、Go 开发。

将 开发打成咖啡，才发现，二者都是 kf 的缩写。

yihong 的 PyCon 的主题演讲中有个 Python for Friends，那么想来，这些年在技术社区和技术产品运营的过程中，还是认识了不少不错的人，我喜欢 nebula 的研发们，yee 老师、四王、甘师傅、mingquan、muniao…，也喜欢不少用户和贡献者，shixiang 和 yihong、ming sir、大叶老师、qiuxuan…每一个 ID 都有一段记忆时光，人真是了不起的生活，能索引如此多的记忆。上周和 muniao 聊天，说起前司的某个开发，我竟然不记得他的名字叫什么，虽然略感尴尬，最后还是问到了他的名字，怪不好意思的。忘记了 ID，却记得 ID 背后的故事，也是人神奇的地方。这里的名字和 ID 还不等同，因为经常见网名，就自动替换名字为网名，等用到名字的时候，发现，诶，我忘记了他/她叫什么，只记得了个网名。上半周看的 Uber 的写放大大概就是解决这种问题吧，虽然数据冗余而庞大，却能记得来龙去脉，人为啥不引入这种放大机制呢（胡说什么呢？Uber 不因为这个要替换掉 PostgreSQL 么？现在的人脑机制才是最优的！——哦？是么？

跑吗？骑吧。

定了个 7 点 10 分的闹钟，本想着去跑步，昨晚突然好奇作为一项运动，跑步和骑行，在收益方面有什么共同点和差异。Gemini 说，二者都能锻炼心肺——是我想要的——能起到强化下肢肌肉的作用——行吧，不是我本意——能改善心理状态——噢，还不错哟——提高耐力，生活会更有活力——是么？——燃烧卡路里——能换成燃烧番茄的就更好了，差别就是跑步锻炼全身，骑行上半身相对静止，所以能锻炼的是下半身。此外，跑步能提高骨密度。一比较，那就去跑步吧，闹钟响了，我也醒了，但腰太疼了，就作罢了，又定了一个 7 点 40 分的闹钟给小猫咪们放饭。番茄又一次成功地占据了我的床中央，我以疼痛的姿势醒来。等放完饭，想着反正骑行的起床时间也是 7 点 40 分，在小猫咪们开饭之后，现在也是，就索性决定今天出去骑行。以后如果跑步的闹钟没起来，就改为骑行。

跑步和骑行，前者让我看不大到风景，仅有一次的跑步之旅，我异常专注于跑步这件事，不要张嘴呼吸，看路况，全然没有骑行那种悠闲，看看沿途变化的心情。大概还是个门外汉，所以没法兼顾风景吧。推车出门的时候刚好 8 点 20 分，混到了上班的车流中，等一个红绿灯，旁边有个小哥刚好也骑着车。他斜挎着包，我两手空空，手机没带，钥匙在兜里。等绿灯开始亮起的时候，我们俩都开始蹬脚踏板，在一个拐弯处，同时拐进了小道。我不想和大车流拥挤在单车道，他我不知道，后面看他的路线应该是去了字节上班（字节就在家门口一公里不远处）。无论我是骑小电动，还是骑自行车，每当在路上看见人骑着非共享单车上班，对方总让我产生一种超然感，这回也不例外，这个小哥也给了我一种超然物外的感觉，尤其他的头发还挑染了一撮绿。这种超然感大概是来源于公路自行车带来的轻松、省力，没有共享单车那种扛哧扛哧的艰难感。小哥很快就离开了小道，我跟在一辆小电动后面骑着，没多久，这个骑着小白车的小姐姐也左拐走了，我一条道骑到底，大概能合并到某个主干道。这是一条我没起骑过的路，路的尽头是一个分岔路，从我身边掠过的电动车们都左拐去了主干道，我不想左拐，就顺拐去右侧，沿着一个类似国道的大马路骑着，路上遇到了两个跑步的人，是跑着而不是走着，路过他们的时候，心里默默地说了句：加油。这是新体验，以往我只会感叹一句：哦，跑步呢？自从上周第一次跑步，以及才跑了 700 米之后，我发现跑步这件事，真难啊，哪些坚持在跑的人，真了不起，加油。就这样沿着不认识的路骑着，看了下码表，2.3km，也没骑多久，但是觉得累了，似乎一阵子不锻炼，耐力变差了。好在坚持到了这条路的中点，它汇集到了一条辅路，这条辅路是我无意间走过的，emmm，就在我家小区楼下的主路旁。所以，我兜了一圈，又要回小区了，那就回吧。这次不在规划中的路线，一共是 3.8 公里，骑了多久忘记了。8 公里的骑行，和 2 公里的跑步一样，变成了锻炼的目标。
