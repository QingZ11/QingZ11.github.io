事情的起源可能是昨天 yajun 突然来微信问我，最近在忙啥，社交上面很久没更新了。

是啊，最近在忙什么呢？即便是数着失业第 N 天过活，也没有立即去找工作的想法。一则，工作的确让我不快；二则，目前的存款，虽然就几万，但还足以支持我继续宅着，不需要找一份工。

所以，目前的状态就是：**没有收入的活着，生活着**。——这里加了 md 样式，纯粹是为了看折腾了快 1 天，合计 6、7 个小时的主题渲染 md 到底是怎么个样子。

## 关于 hugo

因为样式问题，去请教了羽毛熊 html 是否区分变量的大小写，比如：baseURL 和 BaseURL 是 2 个变量。他不知道，我更不知道。只能去瞎折腾，折腾了 3 个多小时，总算是把服务器端的 url 生成问题解决了，只使用了一个 `hugo` 命令，放弃了 `hugo server`，大概后者生成的是 `localhost:1313` 之类的。

这里充分体现了我的不求甚解。

### 我想要的 hugo 主题

找了半圈 hugo 主题，本来想为我们家小猫咪们选一个更合适图片呈现的样式，结果找了一圈，并没有合适的，无意间发现了这个 GitHub 风格的主题，于是就用了，于是就花了 6、7 个小时来解决样式无法呈现的问题。

## 每日一照

昨天 yajun 来问，最近在干嘛。让我想起了，其实有些人似乎不怎么和你往来，但是偶尔有一天，会想起来你，然后在可触达的地方（社交媒体）来找寻你的痕迹。但是宅在家，似乎没有输入的情况下，人也懒得输出，就像给 mona 发的消息，她几乎不回，让我觉得我说的话是个引不起她回复兴趣的话。这不是她的问题，是我的。交流的两个人，话题没有共鸣的花，那自然是消息只是单向请求，而宅在家，没有进行社会化的我，话题的共鸣性更弱了。不同于其他人，但只是说我个人，大抵是个很难引起他人兴趣的人。（并非自怨自艾，这只是一种客观事实）

所以，为了那些偶尔想起来“你我”的人，保持一定的内容更新，我称之为每天一照的东西，可能是非常有必要的。

这是一锅关东煮，四月从日本回来之后，一直想去吃关东煮。无论是全家的，或是 711 都可以。只是一个念头：想要吃关东煮。
